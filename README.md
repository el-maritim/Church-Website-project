# Church Website Project

A responsive church website built with HTML, CSS, and JavaScript, featuring modern design elements and interactive components.

## Features

- Responsive design for mobile and desktop
- Interactive elements with animations (AOS)
- Image galleries and carousels
- Contact forms and event listings
- Bootstrap framework for styling
- Custom fonts and icons

## Technologies Used

- HTML5
- CSS3 (with SCSS)
- JavaScript (jQuery)
- Bootstrap 4
- AOS (Animate On Scroll)
- Owl Carousel
- Magnific Popup
- Font Awesome / Flaticon icons

## Installation

1. Clone the repository:
   ```
https://github.com/el-maritim/Church-Website-project.git  

2. Navigate to the project directory:
   ```
   cd Church-website-project
   ```

3. Open `index.html` in your web browser or serve with a local server.

## Usage

- Customize the content in `index.html`
- Modify styles in `scss/style.scss` and compile to CSS
- Add images to the `images/` folder
- Update JavaScript functionality in `js/main.js`
